# 📦 Inventory Management System

This project is a simple web-based inventory management system developed to practice backend development using Python and Django, while also solving a real-world need for product organization in a small store.

## 🚀 Features

- Product registration
- View available products in stock
- Update product quantities
- Store basic product information such as price and product code

## 🛠 Technologies

- Python
- Django
- SQLite
- HTML
- CSS

## 📚 Project Goal

The goal of this project is to practice web development with Python and Django while building a practical tool for managing product inventory.

## ▶️ Running the Project

Clone the repository:

```bash
git clone https://github.com/your-username/repository-name.git

Navigate to the project folder:

cd repository-name

Create a virtual environment:

python -m venv venv

Activate the virtual environment (Windows):

venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt

Run migrations:

python manage.py migrate

Start the development server:

python manage.py runserver

Open in your browser:

http://127.0.0.1:8000
